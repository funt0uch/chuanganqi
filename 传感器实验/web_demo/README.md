# 传感器工具箱 - Web演示版

## 简介

这是一个基于HTML/CSS/JavaScript的Web演示版本，可以在浏览器中模拟和可视化传感器功能，无需实际手机设备。

## 使用方法

### 方法1：直接打开（最简单）

1. 双击打开 `index.html` 文件
2. 浏览器会自动打开并显示应用界面

### 方法2：使用本地服务器（推荐）

如果直接打开遇到问题，可以使用Python启动本地服务器：

```bash
# Python 3
cd web_demo
python -m http.server 8000

# 然后在浏览器访问
# http://localhost:8000
```

或者使用Node.js：

```bash
# 安装http-server（如果还没有）
npm install -g http-server

# 启动服务器
cd web_demo
http-server -p 8000

# 然后在浏览器访问
# http://localhost:8000
```

## 功能说明

### 1. 加速度传感器
- 实时显示X、Y、Z三轴加速度值
- 点击页面任意位置模拟摇动
- 摇动时会显示提示动画

### 2. 指南针
- 显示旋转的指南针
- 实时显示方向和方位角
- 显示磁场强度

### 3. 光线传感器
- 显示光线强度数值（Lux）
- 可视化光线强度条
- 根据强度显示状态描述

### 4. 陀螺仪
- 显示X、Y、Z三轴角速度
- 模拟设备旋转数据

### 5. 方向传感器
- 显示方位角、俯仰角、翻滚角
- 模拟设备姿态变化

## 注意事项

1. **这是演示版本**：数据是模拟生成的，不是真实传感器数据
2. **仅用于展示**：帮助理解应用界面和功能
3. **真实功能**：需要在Android设备上运行Python版本才能获取真实传感器数据

## 与Python版本的区别

| 特性 | Web演示版 | Python版本 |
|------|----------|-----------|
| 数据来源 | 模拟数据 | 真实传感器 |
| 运行环境 | 浏览器 | Android设备 |
| 功能完整性 | 界面演示 | 完整功能 |
| 部署方式 | 打开HTML | 打包APK |
