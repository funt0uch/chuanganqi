# Conda环境配置说明

## 环境信息

- **环境名称**: PyApp_env
- **Python版本**: 3.9
- **创建日期**: 2024

## 已安装的包

### 核心依赖
- **kivy** (>=2.1.0): GUI框架，用于构建移动应用界面
- **plyer** (>=2.1.0): 跨平台传感器访问库

### 自动安装的依赖包
- Kivy-Garden: Kivy扩展包管理器
- kivy-deps.angle: Kivy的ANGLE依赖
- kivy-deps.sdl2: Kivy的SDL2依赖
- kivy-deps.glew: Kivy的GLEW依赖
- pywin32: Windows平台支持
- docutils, pygments, requests等工具包

## 使用方法

### 激活环境
```bash
conda activate PyApp_env
```

### 运行项目
```bash
python main.py
```

### 退出环境
```bash
conda deactivate
```

## 注意事项

1. 确保在运行项目前已激活PyApp_env环境
2. 桌面环境下传感器功能可能无法正常工作，需要在Android设备上运行
3. 如需重新安装依赖，可以使用：
   ```bash
   pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
   ```

## 环境位置

环境安装在: `E:\Conda\Anaconda3\envs\PyApp_env`
